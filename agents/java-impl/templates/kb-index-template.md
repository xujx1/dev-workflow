## 知识库全景（不要一次全读）

```
app-knowledge-base/
├── _index.md                    ← 你在这里（L0，必读）
├── 00_概览.md                   ← 系统定位、高频接口、QPS 数据（L1）
├── 01_业务与领域知识层.md         ← 领域模型、枚举、术语表（L1）
├── 02_架构与设计层.md             ← DDD/MVC分层、接口清单、表设计（L1）
├── 03_核心流程与逻辑层.md         ← 核心链路、MQ、定时任务、状态机（L1）
├── 04_工程与规范层.md             ← 异常体系、编码规范、配置项（L1）
├── 05_演进与决策记录层.md         ← ADR、技术债务（L1，按需）
│
├── api-docs/                    ← 接口深度文档（L2，按接口编号钻取）
│   └── {NN}_{接口名}.md
│
├── biz-knowledge/               ← 业务流程知识（代码视角）
│   └── modules/
│       ├── 00_导航.md           ← 业务模块索引（L1）
│       └── {NN}_{模块}.md       ← 具体模块（L2）
│
└── test-knowledge/              ← 测试知识库
    ├── 测试知识库导航.md          ← 测试模块索引（L1）
    ├── modules/{NN}_{模块}_测试.md ← 具体模块测试要点（L2）
    └── api-testcase/            ← 接口测试用例（按需，非默认产出）
        └── {NN}_{接口名}_testcase.md
```
