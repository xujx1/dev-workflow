| Redis | Mock / Stub |
| 事务 | `N/A` |

## 7. AI-SDD 测试执行

1. `testcode-gen-agent` 基于本 spec 生成 Java/JUnit 测试代码与 `test_file_list`
2. `tdd-test-runner-agent` 按清单运行测试并生成覆盖率报告
3. 若覆盖率未达 §3.1 目标，**优先**由 `tdd-test-spec-agent(task=supplement-coverage)` 局部补充本 spec，再重新走步骤 1～2；避免默认多轮只改测码、反复跑 Runner
